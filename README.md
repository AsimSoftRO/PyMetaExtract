# PyMetaExtract: Digital Metadata Exfiltrator
<img width="1024" height="1024" alt="PyMetaExtract" src="https://github.com/user-attachments/assets/8047be61-0fe4-4f63-9fa2-574c376d8f5f" />

**PyMetaExtract** is a powerful Python-based OSINT tool designed to crawl websites, identify media assets, and exfiltrate hidden metadata (EXIF). Whether you are a security researcher performing reconnaissance or a privacy-conscious user auditing your own digital footprint, PyMetaExtract provides a deep dive into the "invisible" data attached to files.
